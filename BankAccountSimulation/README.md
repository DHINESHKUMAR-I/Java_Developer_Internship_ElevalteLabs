# Bank Account Simulation

## Objective
Simulate basic banking operations using Java Object-Oriented Programming concepts.

## Features

- Create Savings or Current Account
- Deposit Money
- Withdraw Money
- Display Account Details
- Check Account Balance
- View Transaction History
- Method Overriding
- Inheritance
- Encapsulation

## Technologies Used

- Java
- VS Code
- Terminal

## Concepts Used

- Classes & Objects
- Inheritance
- Method Overriding
- Constructors
- Encapsulation
- ArrayList
- Scanner

## Project Structure

```
Account.java
SavingsAccount.java
CurrentAccount.java
BankSimulation.java
README.md
```

## How to Run

Compile

```
javac *.java
```

Run

```
java BankSimulation
```

## Outcome

This project simulates real-world banking operations using Java OOP principles such as inheritance, encapsulation, and method overriding.
