
###  TASK - 1
# Java Console Calculator

A simple console-based calculator built using Java as part of **Task 1** of the **Elevate Labs Java Developer Internship**.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Menu-driven interface
* Continuous calculations until exit

## Technologies Used

* Java (JDK)
* VS Code / IntelliJ IDEA

## How to Run

```bash
javac Calculator.java
java Calculator
```

### TASK - 2

# Student Record Management System

A simple console-based **Student Record Management System** built using Java as part of **Task 2** of the **Elevate Labs Java Developer Internship**.

## Features

* Add Student
* View Student Records
* Update Student Details
* Delete Student Records
* Uses ArrayList for data storage
* Menu-driven interface

## Technologies Used

* Java (JDK)
* VS Code / IntelliJ IDEA

## How to Run

```bash
javac Student.java StudentManagementSystem.java
java StudentManagementSystem
```
