# Java GUI - ToDo App

## Objective

Build a simple To-Do List application using Java Swing.

## Tools Used

- Java
- Swing
- IntelliJ IDEA Community Edition

## Features

- Add Tasks
- Delete Selected Task
- Clear All Tasks
- Scrollable Task List
- User-friendly GUI

## How to Run

Compile

```bash
javac ToDoApp.java
```

Run

```bash
java ToDoApp
```

## Concepts Used

- JFrame
- JPanel
- JButton
- JTextField
- JList
- JScrollPane
- ActionListener
- Layout Managers
